# The Simpsons Wiki

**The Simpsons Wiki** is a frontend web application that displays detailed information about characters from _The Simpsons_ television series, explore character profiles including **status**, **occupation**, **gender**, and more.

Utilizing _The Simpsons API_, discover everything you need to know about the world's most iconic yellow family.

## Tech Stack

- **React.js:** A JavaScript library for building user interfaces with reusable components.
- **React Router Dom:** Declarative routing for React applications.
- **Vite:** Next-generation frontend build tool for fast development and optimized production builds.
- **CSS:** Custom styling with modern CSS features for responsive design.
- **JavaScript:** Core programming language powering the application logic.

## Features

- **Character Data Fetching:** Retrieve comprehensive character information from The Simpsons API.
- **Pagination:** Load additional characters seamlessly with pagination controls.
- **Interactive Animations:** Smooth hover animations and transitions for enhanced user experience.
- **API Integration:** Dynamically fetch and display data using the public Simpsons API.
- **Responsive Design:** Fully responsive layout optimized for all device sizes.
